# 業界・職種リアル比較ガイド

OpenWorkの口コミデータをもとに、業界・職種の年収・残業・休日・年収カーブを可視化した就活・転職支援Webアプリです。

## 概要

- **対象**：関関同立以上の就活生（大学2〜3年生）および転職検討中の社会人
- **データ出典**：OpenWork（口コミ）、厚労省就労条件総合調査（2025年）、各社有価証券報告書
- **前提**：大手企業・大企業前提のデータ

## 機能

### 就活版
- **業界の概要**：8業界のカード表示・詳細パネル（代表企業・メリット・デメリット・会社間比較）
- **数値で比較**：年収・残業・休日・有給消化率の一覧・ソート機能
- **年収カーブ**：22〜50歳の業界別年収推移グラフ（業界選択可）
- **年収だけじゃない**：福利厚生・会社差・OpenWorkの使い方
- **職種ガイド**：6職種の詳細（離職率・リモート率・リアルな声・離職の実態）
- **価値観診断**：6軸スライダーによる向いている業界レコメンド

### 転職版
- **年収シミュレーション**：現年収・年齢を入力して業界別年収との差を可視化
- **ポジショニングマップ**：年収×残業時間の散布図
- **OpenWorkスコア分析**：5項目スコアの業界別比較
- **キャリアパス**：業界間転職パターンと年収変化目安

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```

## Vercelへのデプロイ

1. このリポジトリをGitHubにpush
2. [Vercel](https://vercel.com)でリポジトリをインポート
3. Framework Preset: **Next.js** を選択
4. Deploy ボタンをクリック

環境変数の設定は不要です（外部APIは使用していません）。

## データについての注意

- 年収データはOpenWorkの口コミ平均を参照しており、実際とは異なる場合があります
- 同じ業界でも会社・部署・職種により大きく異なります
- 数値は目安であり、保証するものではありません
